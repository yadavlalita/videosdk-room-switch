# VideoSDK Room Switching with Media Relay

## Tech Stack
- React.js
- Node.js
- VideoSDK

## Features
- Join Room A
- Normal Room Switching
- Media Relay between rooms
- Secure token generation

## Project Structure
frontend/ - React app  
backend/ - Token server

## Setup (Local)
1. cd backend
2. npm install
3. add .env
4. npm start

5. cd frontend
6. npm install
7. npm start
